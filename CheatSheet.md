# If-test
## bare if
```python 
alder = 20

if(alder > 17 ):
    print("Du er gammel nok til å kjøre bil")

# alternativ med større eller lik 18
if(alder >= 18 ):
    print("Du er gammel nok til å kjøre bil")
```
## if med else

```python 
alder = 15

if(alder > 17 ):
    print("Du er gammel nok til å kjøre bil")
else:
    print("Du må nok sykle enn så lenge")

```

```python 
alder = 15

if(alder > 17 ):
    print("Du er gammel nok til å kjøre bil")
elif(alder > 15):
    print("Du er gammel nok til å kjøre moped")
else:
    print("Du må nok sykle enn så lenge")

```
